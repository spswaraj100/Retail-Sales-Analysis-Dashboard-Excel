# Retail-Sales-Analysis-Dashboard-Excel
Interactive retail sales analysis dashboard created using MS Excel, showcasing sales trends, top states, customer insights, and order performance.

## Project Overview
This project analyzes retail sales data and presents insights using an interactive dashboard built in MS Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Dashboard Insights
- Monthly Sales vs Orders trend
- Top 5 performing states by sales
- Orders distribution by age group and gender
- Sales comparison between men and women
- Order status analysis
- Sales channel performance

## Key Highlights
- Women contribute higher sales (64%)
- Maharashtra is the top-performing state
- Majority of orders are delivered successfully (92%)
- Amazon is the leading sales channel

- Data Cleaning
- Data Analysis
- Dashboard Design
- Business Insights
